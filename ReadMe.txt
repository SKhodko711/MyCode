Переключение между окнами
Имя входного файла:	input.txt
Имена выходных файлов:	output1.txt, output2.txt, output3.txt
При работе в ОС Windows часто запущено несколько приложений. Каждое из приложений работает в отдельном окне. 
Для переключения между окнами используется комбинация клавиш «Alt+Tab». 
Чтобы переключиться в другое окно, можно нажать клавишу «Alt» и затем, не отпуская ее, несколько раз нажать клавишу «Tab».
Пусть запущено n приложений. ОС рассматривает список как циклический, переходя после последнего элемента списка к первому.
При запуске нового приложения оно добавляется в список перед активным приложением.
При переключении между приложениями выбранное приложение устанавливается в начало списка запущенных приложений, остальные приложения при этом остаются на месте.
Задана последовательность действий пользователя, где каждое действие – либо запуск приложения, либо переключение между окнами.
Формат входного файла
Файл содержит строки описания действий пользователя.
Запуск приложения описывается строкой «Run <имя приложения>». Здесь <имя приложения> - строка из не более чем 100 латинских букв, цифр и пробелов. Она отделена от слова Run хотя бы одним пробелом. Прописные и строчные буквы считаются различными.
Переключение между приложениями описывается строкой «Alt+Tab+…+Tab». Здесь подстрока «+Tab» повторена в точности столько раз, сколько раз пользователь нажал клавишу «Tab», не отпуская клавишу «Alt».
Также есть команды delete
Формат выходных файлов
В 1 выходной файл вывести список запущенных приложений в порядке возрастания кодов символов
В 2 выходной файл вывести список переключений между приложениями в порядке возрастания количества подстрок «Tab» 
В 3 выходной файл вывести последовательность имен приложений, с которыми работал пользователь в порядке, в котором их окна становились активными 
Вывести удалённые
